🏠 House Price AI

Smart Property Valuation using Machine Learning

📌 Project Overview

House Price AI is a machine learning web application that predicts property prices based on features such as area, bedrooms, location, amenities, and construction details. It helps users estimate realistic property values instantly.

⚡ Key Features

🤖 AI Price Prediction
📊 Market Insights
📜 Prediction History
👤 User Dashboard
🛠 Admin Dashboard
🔔 Real-time Notifications

🧠 Machine Learning Models

The system uses trained regression models to analyze housing data and estimate property prices accurately.

  • Random Forest Regressor
  • Gradient Boosting Regressor
  • Feature Engineering for property attributes

💻 Technology Stack

Python Flask Machine Learning HTML CSS JavaScript Chart.js SQLite

🎯 Project Goal

The goal of this project is to provide an intelligent system that helps buyers, sellers, and real estate investors estimate property prices quickly using machine learning.
